Order Service

GET /orders to see all the orders.

GET /orders/{id} retrieves an order from the DB using the order ID.
PUT /orders/{id} updates an order using an order ID.
DELETE /orders/{id} removes an order from the DB using an order ID.
POST /orders creates a new order.
